= charAt()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Access a particular character of the String.

[%hardbreaks]


[float]
=== Syntax
`myString.charAt(n)`


[float]
=== Parameters
`myString`: a variable of type `String`. +
`n`: a variable. Allowed data types: `unsigned int`.


[float]
=== Returns
The n'th character of the String.
